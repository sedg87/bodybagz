# 🩸 BodyBagz

A horror news and review blog covering movies, TV, and video games.

## 🔗 Live Site
[bodybagz.co.uk](https://bodybagz.co.uk)

## 🛠️ Built With
- Python / Flask
- SQLite
- HTML / CSS
- Jinja2

## ✨ Features
- Full admin panel with post management
- Image uploads
- Movie, TV and game reviews
- Coming soon / upcoming section
- Contact form with rate limiting
- View tracking and trending posts
- Featured posts
- Search functionality
